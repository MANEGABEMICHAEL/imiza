# Familles Militaires Congolaises

Plateforme de soutien psychologique, de formations en ligne et de collecte de fonds pour les familles militaires congolaises.

## 🎯 Mission

Soutenir et accompagner les familles militaires congolaises à travers des services psychologiques spécialisés, des formations professionnelles et une communauté solidaire pour faire face aux défis uniques de la vie militaire.

## 🧠 Services Psychologiques

- **Thérapie Individuelle** : Sessions confidentielles avec des psychologues spécialisés dans le trauma militaire
- **Thérapie Familiale** : Renforcement des liens familiaux et gestion des tensions
- **Groupes de Soutien** : Partage d'expériences dans un environnement bienveillant
- **Support d'Urgence** : Disponible 24/7 pour les situations critiques

## 📚 Formations en Ligne

## 🚀 Démarrage Rapide

### Prérequis
- Node.js (version 18 ou supérieure)
- npm ou yarn

### Installation

1. Cloner le dépôt :
```bash
git clone <repository-url>
cd familles-militaires-congolaises
```

2. Installer les dépendances :
```bash
npm install
```

3. Démarrer le serveur de développement :
```bash
npm run dev
```

4. Ouvrir votre navigateur et naviguer vers `http://localhost:5173`

### Scripts Disponibles

- `npm run dev` - Démarrer le serveur de développement
- `npm run build` - Construire pour la production
- `npm run preview` - Prévisualiser la build de production
- `npm run lint` - Exécuter ESLint
